# PRODUCT REQUIREMENTS (PRD)

Last updated: October 18, 2025 17:25

## Target Users
- English-speaking retail traders seeking fast insights and discovery.

## MVP User Stories
- As a user, I see extreme movers and categories on the landing without logging in.
- As a user, I can run preset scouters (bullish, bearish, gaps, coin stocks, meme, sectors) and view top 10.
- As a user, I can search a symbol quickly from the top bar.
- As a user, I can save symbols to my lists and revisit them.

## Non-Goals (MVP)
- No brokerage integration or order routing.
- No guaranteed real-time tick-by-tick across all symbols.

## KPIs
- 1-day and 7-day retention
- Sessions per user per day
- Rewarded ad completion rate
- Time to first signal (TTFS)
